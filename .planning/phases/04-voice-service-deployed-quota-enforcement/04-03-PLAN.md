<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator -> deployed /api/offer (smoke credential) | the smoke path authenticates with a bypass credential; its blast radius must be bounded |
| public internet -> deployed task (UDP media) | the smoke exercises the exact untrusted-media path a real browser will use |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-05 | Spoofing / abuse | Smoke service credential leaking or over-scoped, bypassing quota broadly | high | mitigate | The credential only sets bypass_accounting; it still traverses /api/offer and the SG; it is an SSM high-entropy secret never printed by kv; 04-04 honors the bypass as accounting-only (never auth-only) (Task 1) |
| T-04-16 | Denial of service | A stuck smoke session holding a slot / task | low | mitigate | kv smoke has a bounded ICE timeout and always closes the PeerConnection on exit; bypass sessions still hit the D-06 idle teardown once 04-05 ships (Task 1) |
| T-04-SC | Tampering | go get github.com/pion/webrtc/v4 | low | mitigate | pion/webrtc is the standard, actively-maintained Go WebRTC library (verify on pkg.go.dev + github.com/pion/webrtc); Go modules are outside the pip/npm/cargo legitimacy-gate scope; pinned in go.mod/go.sum with checksum verification |
</threat_model>
