| Boundary | Description |
|----------|-------------|
| external media (RTP/μ-law bytes) → `parse_rtp`/`ulaw_decode` | The one new untrusted-input surface. Offline-fed this phase (`OfflineRtpMediaSession`); socket-fed Phase 11. Truncated/malformed/adversarial datagrams must be rejected (None) or decoded totally, never raise. |
| telephony media package → shared runtime | ZERO crossings this plan: no `call_runtime`/`pipeline`/`factories` edit, no provider construction, no new spend path. Telephony is a new leaf module. |
