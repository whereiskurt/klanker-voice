## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| unauthenticated request → /admin | Only ADMIN_EMAILS operators may see any admin content; others must not even learn the route exists |
| transcript text → rendered HTML | Speech is untrusted input; a prompt-injected utterance must not execute as markup (stored XSS via speech) |
| auth task role → ledger bucket | Read-only; the report must not be able to write or delete ledger objects |
