## Phase Goal

**As** Kurt (the operator), **I want to** regenerate KPH's knowledge packs from my repos
with one deliberate command that refuses non-public sources and flags anything unsafe to
say, **so that** refreshing what KPH knows is a reviewed script run, never a risky manual edit.

<objective>
Deliver the D-07 manual refresh path (ROADMAP criterion 3): an offline distillation script
(`apps/voice/scripts/refresh_knowledge.py`) that reads the curated manifest, surveys each
public source (repos + optional transcript) via same-vendor Anthropic calls, distills
voice-friendly per-topic packs + the style layer (map-reduce shape), runs a do-not-say
scrubber over every output, and writes into the tracked `knowledge/` tree for a git-diff
review. Expose it via `make -C apps/voice knowledge` and a thin `kv knowledge refresh`
dispatcher. The scrubber is the phase's enforced security control (Pitfall 4 / D-02).

Purpose: Makes freshness reproducible and safe. This is additive to the runtime loop —
the script never runs live (D-07); it only produces reviewable content.
Output: The refresh script + importable scrubber + operator commands + refresh tests.
</objective>
