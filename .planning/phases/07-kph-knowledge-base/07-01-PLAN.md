<objective>
The km walking slice: the thinnest end-to-end path where a real visitor asks about
klanker-maker and KPH answers correctly, in Kurt's voice, from a two-block cached system
prompt — the router picks the topic, the ack fires only on a genuine deep-pack switch, and
prompt caching engages (cache_read_input_tokens > 0). Every downstream plan (retrieval, more
topics, refresh, evals) extends this seam.

Purpose: Prove the whole architecture on ONE topic before scaling — MVP vertical slice, not a
horizontal foundation. Activates PIPE-10 (as router + cached packs, not retrieval infra),
extends PIPE-06 (persona + STYLE layer in the versioned system prompt), honors PIPE-07 (the
router is keyword-first + same-vendor Haiku fallback — no 4th vendor).
Output: `[knowledge]` config + manifest + topic-map + km deep pack + Kurt STYLE layer +
two-block prompt assembly + KnowledgeRouterProcessor + advisory lint + km eval + live
cache-verify.
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| spoken utterance → router | Untrusted public-mic text drives topic selection (and a possible fallback LLM call). |
| corpus files → cached system prompt | Curated, public-by-design content becomes world-readable prompt (assume full prompt extraction). |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-07-01 | Information Disclosure | km deep pack / style layer content | medium | mitigate | Corpus is public-by-design (D-02); advisory_lint flags account-ids/ARNs/internal-hostnames; the promoted km-digest carries a curated do-not-say section; git-diff review (D-09) before commit. |
| T-07-02 | Tampering | prompt-injection via spoken utterance | low | accept | Nothing secret is in the prompt by design (D-02); persona ground rules already treat user speech as conversation, never configuration. |
| T-07-03 | Spoofing/Correctness | router picks the wrong topic pack → confident wrong answer | medium | mitigate | confidence_floor enforced; below it, same-vendor Haiku fallback or which-did-you-mean deferral (Pitfall 1); never guess. |
| T-07-04 | Reputational | KPH volunteers crude/edgy humor to a stranger on a public mic | high | mitigate | PG-13 + match-and-escalate guardrail authored into the cached style layer; verified by an eval in 07-05. |
| T-07-SC | Tampering | package installs | low | accept | This phase installs NO new packages (RESEARCH: reuses anthropic 0.116.0 + PyYAML 6.0.3). No legitimacy gate required. |
</threat_model>
