| Boundary | Description |
|----------|-------------|
| ARI StasisStart event → controller | Caller-influenced fields (ANI/DID); a hostile inbound call reaches here |
| controller resource allocation → Asterisk (bridge/external channel/socket) | Every allocation is a resource that must be torn down on any exit |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-11-05-01 | Denial of service | resource leak on abnormal exit | high | mitigate | Single idempotent `_close_active_call` funnels ChannelDestroyed + hard-timeout + quota-denied; ActiveCall.closed guard prevents double/partial teardown (Task 2 + §16 tests) |
| T-11-05-02 | Denial of service | silent open PSTN call | high | mitigate | lifecycle.on_released composed with ARI hangup of the SIP channel — a hard timeout always hangs up (Task 2, §17) |
| T-11-05-03 | Elevation of privilege | quota-denied caller reaches pipeline | high | mitigate | quota.start_gate rejection constructs NO CallSession and tears down the gate bridge (R6); concurrency=1 enforced |
| T-11-05-04 | Information disclosure | structured logs | medium | mitigate | Call id logged; SIP passwords / PIN / passphrase never logged (§13) |
