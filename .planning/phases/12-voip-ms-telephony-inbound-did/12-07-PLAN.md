<objective>
Stand up the minimum secure `telephony-edge` (D-01): a deployed, SSM-backed, inbound-only Asterisk edge on Fargate with a public egress IP for the registration trunk, a security group locked to the eight Toronto VoIP.ms POP IP ranges (never open to the internet), ARI private-network-only, and full SSM `valueFrom` secret wiring (D-04) for VOIPMS_SIP_*, ASTERISK_ARI_*, TELEPHONY_ACCESS_PIN/PASSPHRASE_WORDS, and the `/tel` endpoint auth token. Purpose: The §19-D exit criterion ("public DID reliably reaches Klanker from the cellular network") is only testable against a real reachable edge with secrets consumed via valueFrom. This provisions that edge. Requires operator credentials/approval — `autonomous: false`.
Output: telephony-edge service stub, a POP-locked security group, an Asterisk edge Dockerfile, and a live deploy.
</objective>
